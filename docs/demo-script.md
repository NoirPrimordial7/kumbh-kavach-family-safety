# Jury demo sequence

1. Open the welcome screen and select **Start Jury Demo**.
2. Show the four named team devices: Ashwin Gudur, Aditya Gholap, Himank Maheshwari, and Arya Dhumal.
3. Open safety ranges and change the safe radius from 50 m to 20 m.
4. Select Arya and use **Move away**.
5. Show the warning transition, then advance beyond the grace period to separation.
6. Select **Simulated SOS** and explain that no real emergency service is contacted.
7. Start Reunion Mode and select a meeting point on the map (or use the guardian position).
8. Move Ashwin and Arya toward the point.
9. Use **Enter reunion radius** and show the configured radius and progress.
10. Use the simulated/manual confirmation after stable proximity and show reunited.
11. Switch the browser network offline.
12. Continue moving the simulated device and show that the coordinate fallback, circles, markers, and local state remain operational.
13. Open `/simulation` and demonstrate that the preserved crowd-scale research layer is separate from the private family product.

Reset All returns configuration, speed, random movement, alerts, and scenario to recommended deterministic defaults. REAL NOW, SIMULATED NOW, and FUTURE NATIVE labels should remain visible when explaining capability boundaries.
